## 🐶🐱 프로젝트 소개

- 이게 멍냥은 반려동물의 피부질환과 안구질환을 AI를 활용한 사진 분석으로 진단하는 서비스를 메인 기능으로 갖는 웹 서비스입니다. <br>
- 반려 동물의 여러 병들에 대한 증상 별 응급 처치 정보를 제공합니다 <br>
- 반려 동물 자랑과 잃어버린 반려동물을 찾는 커뮤니티 서비스를 제공합니다 <br>
- 사용자 주변의 반려동물 관련 병원과 애완용품상점 등의 정보를 제공합니다 <br>
